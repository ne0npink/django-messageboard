# CloudySky Messageboard

CloudySky is a Django-based messageboard application that supports
user authentication, post creation, inline commenting, and moderation.

---

## Features

- User authentication (login/signup)
- Create posts and comments
- Inline commenting directly on posts
- Post and comment moderation (hide functionality)
- JSON API endpoints for posts and comments
- Server-rendered HTML messageboard view

---

## Tech Stack

- Python
- Django
- SQLite
- HTML / CSS
